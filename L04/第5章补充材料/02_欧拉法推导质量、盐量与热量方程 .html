海水基本守恒方程

基于欧拉控制体法推导质量、盐量与热量方程 (包含矢量形式与展开形式)

0. 欧拉控制体方法

在推导海水的各项守恒方程时，我们采用欧拉方法：在空间中取一个固定不动的微小长方体作为控制体 (Control Volume)。

  • 中心坐标为 (x, y, z)
  • 边长为 \delta x, \delta y, \delta z (或记为 dx, dy, dz)
  • 体积记为 \delta \tau = \delta x \delta y \delta z

推导的核心逻辑是统一的收支平衡原理：

控制体内某物理量的增加率
=
通过表面净流入的该物理量 + 内部源汇

(上方为 3D 控制体微元。内容可向下滚动，支持鼠标拖拽旋转3D视角)

1. 连续方程 (质量守恒)

考察控制体内海水质量的收支。物理量密度为 $\rho$。将三维空间净流入量与控制体内质量增量 $\frac{\partial\rho}{\partial t}\delta\tau$ 平衡，约去 $\delta\tau$ 得到基础连续方程。

\frac{\partial \rho}{\partial t} + \frac{\partial(\rho u)}{\partial x} + \frac{\partial(\rho v)}{\partial y} + \frac{\partial(\rho w)}{\partial z} = 0

🌊 引入包辛尼斯克近似 (不可压缩流体)

海水不可压缩，密度变化极小，方程简化为速度场散度为零：

【矢量形式】

\nabla \cdot \mathbf{V} = 0

【展开形式 (笛卡尔坐标)】

\frac{\partial u}{\partial x} + \frac{\partial v}{\partial y} + \frac{\partial w}{\partial z} = 0

2. 盐扩散方程 (盐量守恒)

考察控制体内盐分的收支。单位体积含盐量为 $\rho S$。盐分的输运包括**流动输运(平流)**和**分子扩散(浓度梯度引起，Fick定律)**两部分。

结合连续方程消去冗余项，令 $k_x=k_y=k_z=k$，可得到：

【矢量形式】

\frac{\partial S}{\partial t} + \mathbf{V} \cdot \nabla S = k_D \nabla^2 S

【展开形式 (笛卡尔坐标)】

\frac{\partial S}{\partial t} + u\frac{\partial S}{\partial x} + v\frac{\partial S}{\partial y} + w\frac{\partial S}{\partial z} = k_D \left( \frac{\partial^2 S}{\partial x^2} + \frac{\partial^2 S}{\partial y^2} + \frac{\partial^2 S}{\partial z^2} \right)

其中 $k_D = k/\rho$ 为分子盐扩散系数。
(3D视图中，加入了黄色的小箭头代表沿梯度方向的分子扩散通量)

3. 传热方程 (能量守恒)

考察控制体内热量的收支。单位体积热量为 $\rho c_p \theta$ (其中 $\theta$ 为位温， $c_p$ 为定压比热)。
海表以下，热量传递主要由**对流平流**和**热传导 (Fourier定律)**完成。

假定 $c_p$ 和传导系数 $k$ 为常数，并结合连续方程简化后得到：

【矢量形式】

\frac{\partial \theta}{\partial t} + \mathbf{V} \cdot \nabla \theta = k_\theta \nabla^2 \theta

【展开形式 (笛卡尔坐标)】

\frac{\partial \theta}{\partial t} + u\frac{\partial \theta}{\partial x} + v\frac{\partial \theta}{\partial y} + w\frac{\partial \theta}{\partial z} = k_\theta \left( \frac{\partial^2 \theta}{\partial x^2} + \frac{\partial^2 \theta}{\partial y^2} + \frac{\partial^2 \theta}{\partial z^2} \right)

其中 $k_\theta = k/(c_p \rho)$ 为分子热传导系数。
(3D视图中，橙色波纹箭头代表热传导过程)

4. 守恒方程的终极统一

对比盐扩散方程与传热方程，我们可以发现自然界中物质与能量输运的数学表达具有惊人的统一性。它们都可以抽象为如下的**对流-扩散方程 (Advection-Diffusion Equation)**：

【当地变化率 + 平流项 = 扩散/传导项 + 源汇项】

\frac{\partial C}{\partial t} + \mathbf{V} \cdot \nabla C = K \nabla^2 C + Q

【完全展开形式】

\frac{\partial C}{\partial t} + u\frac{\partial C}{\partial x} + v\frac{\partial C}{\partial y} + w\frac{\partial C}{\partial z} = K \left( \frac{\partial^2 C}{\partial x^2} + \frac{\partial^2 C}{\partial y^2} + \frac{\partial^2 C}{\partial z^2} \right) + Q
  • 当浓度 $C = 1$ 时，退化为连续方程 (无扩散)。
  • 当浓度 $C = S$ 且 $K = k_D$ 时，即为盐扩散方程。
  • 当浓度 $C = \theta$ 且 $K = k_\theta$ 时，即为传热方程。

💡 深入理解：为什么 C=1 会退化为连续方程？

如果我们写出最原始、最通用的物质通量守恒方程（未做不可压缩简化的状态）：

\frac{\partial(\rho C)}{\partial t} + \nabla \cdot (\rho \mathbf{V} C) = \text{扩散项}(K\nabla^2C) + \text{源汇项}(Q)

现在，我们把**追踪海水自身**的 $C = 1$ 代入这个方程：

  • 时间变化： \frac{\partial(\rho \cdot 1)}{\partial t} = \frac{\partial \rho}{\partial t}
  • 平流输运： \nabla \cdot (\rho \mathbf{V} \cdot 1) = \nabla \cdot (\rho \mathbf{V})
  • 扩散项： 因为 C=1 是个常数，代表空间中到处都是 100% 的海水，没有浓度差，所以常数的导数（梯度）为 0，扩散项为 0。
  • 源汇项： 如果没有外界凭空注水，源汇项 $Q = 0$。

把这些结果拼在一起，原来的通用方程就变成了：

\frac{\partial \rho}{\partial t} + \nabla \cdot (\rho \mathbf{V}) = 0

这正是我们在第 1 步推导出来的连续方程（质量守恒）！

这种高度对称的形式，是物理海洋学和计算流体力学(CFD)能够使用通用算法求解多种物理过程的核心基础。
