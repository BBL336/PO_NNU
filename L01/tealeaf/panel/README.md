# 🍵 茶杯搅拌的物理学演示

这个 Panel 应用演示了搅拌茶杯时的两个有趣物理现象：
1. **水面形成抛物面**：由于离心力作用
2. **茶叶汇聚中心**：由于次级流动效应

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行应用

方式一（推荐，Panel CLI）：

```bash
panel serve tea_cup_physics.py --show
```

应用将在浏览器中自动打开：`http://localhost:5006/tea_cup_physics`

方式二（直接运行 Python 脚本）：

```bash
python tea_cup_physics.py
```

运行后会自动启动本地服务并打开浏览器。

### 常见问题

- 若看到 `RuntimeError: no running event loop`，请确认使用的是最新版本代码（已在 `tea_cup_physics.py` 中修复）。
- 建议先激活 Conda 环境再运行：

```bash
conda activate base
```

## 功能特性

### 交互控制
- **搅拌状态切换**：观察搅拌和静止状态下的差异
- **角速度调节**：改变搅拌速度，观察抛物面曲率变化
- **时间推进**：模拟茶叶运动过程
- **位置重置**：重新随机分布茶叶位置

### 可视化
- **3D视图**：立体展示水面抛物面形状
- **俯视图**：显示茶叶分布和水流方向

### 物理原理
详细解释了：
- 圆周运动与向心力
- 抛物面形成的数学推导
- 次级流动（Secondary Flow）机制
- 茶叶悖论（Tea Leaf Paradox）

## 物理背景

**茶叶悖论**由物理学家爱因斯坦在1926年提出，用于解释为什么搅拌后的茶叶会聚集在杯底中心，而不是被"甩"到边缘。这个现象也能解释河流弯道的泥沙沉积规律。

## 操作建议

1. 启动时默认为搅拌状态，可以看到水面呈抛物面
2. 调整角速度滑块，观察抛物面高度变化
3. 关闭搅拌开关，切换到静止状态
4. 多次点击"更新视图"按钮，观察茶叶逐渐向中心聚集的过程
