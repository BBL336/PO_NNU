🚢 航线与洋流沙盘演示

🌊 全球洋流对航运的影响：
• 黑潮: 为北上及东行的船舶提供"顺流"加速。
• 北印度洋季风: 冬夏风向相反，考验船长对开航时机的把握。
• 厄加勒斯洋流: 与南大洋西风带交汇，好望角因此被称为"风暴角"。
• 北大西洋暖流: 带来温水，保障欧洲西部港口终年不冻。
📍 三条航线途径点：
🔵 苏伊士航线
上海 → 台湾海峡 → 马六甲海峡 → 斯里兰卡 → 曼德海峡 → 苏伊士运河 → 地中海 → 直布罗陀海峡 → 比斯开湾 → 英吉利海峡 → 伦敦

🔴 好望角航线
上海 → 台湾海峡 → 巽他海峡 → 好望角 → 西非 → 比斯开湾 → 英吉利海峡 → 伦敦

🟢 北极航线
上海 → 对马海峡 → 津轻海峡 → 白令海峡 → 维利基茨基海峡 → 巴伦支海 → 北海 → 伦敦
